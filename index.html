Join Gem's Sandbox Discord server!

In this server you'll find a fun (and insane) community. If we get enough members, we'll have enough players for game events. When joining the server, you'll have the option to pick roles for the events you'll get notified for.











Events

Two types of events can happen on the server, Game and Video ones. In our Game events we ping the @Gamers to play an online game, this is usually with homebrewed Nintendo consoles. In our Video events we get the @Viewers here so we can watch YouTube videos in the server with the Discord Watch Together activity.
